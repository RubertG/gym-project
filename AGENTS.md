# IRON TRACK — Agent Bootstrap

## Descripción del Proyecto

Aplicación web de seguimiento de entrenamiento en gimnasio. Permite a los atletas crear rutinas, registrar sesiones de entreno en tiempo real, revisar historial y progreso, y descubrir rutinas compartidas por la comunidad.

## Stack Principal

- Astro 6 (framework principal, SSR/SSG, API Routes)
- React 19 (islas de interactividad)
- Tailwind 4 (estilos y design tokens)
- TypeScript strict
- Supabase (Auth, PostgreSQL, RLS, Storage)
- Zustand (estado global ligero, solo sesión de entreno activa)
